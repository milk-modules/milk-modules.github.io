Navigation

Introduction

Navigating within an app can be challenging for users if the developer utilizes a custom navigation approach. Android Studio includes a project template that provides developers with the ability to build apps that utilize a persistent "navigation drawer". This UI element provides developers with the ability to present a standard navigation mechanism for user to navigate within the app
